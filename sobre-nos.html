Sobre Nós

O Tech Trio é um grupo de alunos da Unidade Paulista da Faculdade de Informática e Administração Paulista (FIAP), cursando o primeiro ano do curso presencial de Análise e Desenvolvimento de Sistemas (turma 1TDSPX).

Marina Yumi Kanadani

Marina é a representante da equipe Tech Trio e responsável pelo Front-End, Banco de Dados e Python no projeto CoralGuard. Como estudante, ela aplica seus conhecimentos adquiridos em sala de aula para criar interfaces de usuário intuitivas e visualmente atraentes. Marina também trabalha no desenvolvimento e manutenção do banco de dados do CoralGuard, garantindo que os dados sejam armazenados e acessados de forma eficiente. Além disso, ela utiliza Python para desenvolver scripts e automações que melhoram a funcionalidade e a performance do CoralGuard.

Rodrigo Santos Freitas

Rodrigo é responsável pelo desenvolvimento em Java e auxílio no Front-End no projeto CoralGuard. Ele utiliza suas habilidades em programação Java para implementar funcionalidades essenciais do sistema, assegurando a robustez e eficiência do código. Além disso, Rodrigo colabora ativamente no desenvolvimento do Front-End, trabalhando em estreita parceria com a equipe para criar interfaces de usuário intuitivas e visualmente agradáveis. Seu comprometimento e conhecimento técnico são vitais para o sucesso do CoralGuard, contribuindo diretamente para a missão de preservação dos recifes de coral.

Gabriela Liborio Souza

Gabriela é a responsável pela gestão de projetos, desenvolvimento do chatbot e auxílio no Front-End no projeto CoralGuard. Como estudante, ela utiliza suas habilidades organizacionais para coordenar as atividades da equipe, assegurando que os prazos sejam cumpridos e que o projeto avance de maneira eficiente. Além disso, Gabriela se dedica ao desenvolvimento do chatbot, uma ferramenta crucial para a interação com os usuários do CoralGuard, e também auxilia no desenvolvimento do Front-End, contribuindo para criar interfaces intuitivas. Sua capacidade de gerenciar projetos e desenvolver soluções tecnológicas inovadoras é essencial para o sucesso do CoralGuard, contribuindo para a proteção e preservação dos recifes de coral.

CoralGuard é uma plataforma dedicada à preservação e monitoramento de recifes de coral. Nossa missão é proteger esses ecossistemas vitais para as gerações futuras, utilizando tecnologia de ponta e promovendo a colaboração entre comunidades e cientistas.
